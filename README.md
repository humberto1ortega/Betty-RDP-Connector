# Betty-RDP-Connector
App made using Autohotkey that sends a wake on LAN magic packet to wake up a local pc then RDPs into it when the pc is online.

Instructions on use:
1)  Unzip the BettyServer.zip folder.
2)  Make sure you have set up your server pc to allow wake on LAN.
3)  Connect to your local PC Via RDP initially to save your credentials.
4)  Run the Betty Server EXE file.
5)  Select the Correct RDP Server from the dropdown.
6)  Fil in the necessary local ip and mac address of the device you are trying to connect to.
7)  Click "Start Server" and the app will wake up your pc, then automatically RDP into it when it is online.
